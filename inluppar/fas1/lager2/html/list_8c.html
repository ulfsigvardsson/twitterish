Inserts a new element at a given index.

If list's copy function is non-NULL, it will be applied to elem and its result stored in the list. Otherwise, elem will be stored in the list.

All indexes are valid. 0 means first element. Negative indexes count backward and too large negative indexes equal 0. Too large positive indexes are same as -1.
